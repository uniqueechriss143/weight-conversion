# BMI & Health Assessment System or Weight Conversion System

Description:
This is a beginner Python project that calculates Body Mass Index (BMI) and gives health recommendations based on the result. It also includes a PAR-Q (Physical Activity Readiness Questionnaire) to assess basic fitness habits.

Features:
- BMI Calculator (kg/lb & meters/inches support)
- Health categories:
  - Underweight
  - Healthy
  - Overweight
  - Obese
- Personalized health recommendations
- PAR-Q health questions
- Input validation system
